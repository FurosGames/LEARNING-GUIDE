# 📦 Módulos (import / export)

Los módulos permiten **organizar el código** en archivos separados, cada uno con su propia responsabilidad.

---

## ¿Por qué usar módulos?

- ✅ **Organización**: código separado por función
- ✅ **Reutilización**: importar funciones en múltiples archivos
- ✅ **Encapsulación**: cada módulo tiene su propio scope
- ✅ **Mantenimiento**: más fácil de encontrar y corregir bugs

---

## Named exports (exportaciones con nombre)

```javascript
// 📁 math.js

// Exportar al declarar
export const PI = 3.14159;

export function sumar(a, b) {
  return a + b;
}

export function restar(a, b) {
  return a - b;
}

export class Calculadora {
  static multiplicar(a, b) {
    return a * b;
  }
}

// O exportar al final
const EULER = 2.71828;
function dividir(a, b) {
  if (b === 0) throw new Error("No se puede dividir por 0");
  return a / b;
}

export { EULER, dividir };
```

### Importar named exports

```javascript
// 📁 app.js

// Importar específicos
import { sumar, restar, PI } from "./math.js";
console.log(sumar(2, 3)); // 5
console.log(PI);           // 3.14159

// Renombrar al importar
import { sumar as sum, PI as pi } from "./math.js";
console.log(sum(2, 3)); // 5

// Importar TODO como un namespace
import * as math from "./math.js";
console.log(math.sumar(2, 3));     // 5
console.log(math.PI);              // 3.14159
console.log(math.Calculadora.multiplicar(3, 4)); // 12
```

## Default export (exportación por defecto)

Cada módulo puede tener **un solo default export**:

```javascript
// 📁 Usuario.js
export default class Usuario {
  constructor(nombre, email) {
    this.nombre = nombre;
    this.email = email;
  }

  toString() {
    return `${this.nombre} <${this.email}>`;
  }
}

// 📁 config.js
const config = {
  apiUrl: "https://api.example.com",
  timeout: 5000,
  debug: false,
};
export default config;
```

### Importar default exports

```javascript
// El nombre de importación puede ser cualquiera
import Usuario from "./Usuario.js";
import miConfig from "./config.js";

const user = new Usuario("Samuel", "samuel@mail.com");
console.log(user.toString()); // "Samuel <samuel@mail.com>"
console.log(miConfig.apiUrl); // "https://api.example.com"
```

## Combinar default + named exports

```javascript
// 📁 api.js
const BASE_URL = "https://api.example.com";

export async function get(endpoint) {
  const res = await fetch(`${BASE_URL}${endpoint}`);
  return res.json();
}

export async function post(endpoint, data) {
  const res = await fetch(`${BASE_URL}${endpoint}`, {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify(data),
  });
  return res.json();
}

// Default: la función principal del módulo
export default { get, post, BASE_URL };
```

```javascript
// 📁 app.js

// Importar default y named al mismo tiempo
import api, { get, post } from "./api.js";

// Usar el default
const users = await api.get("/users");

// O usar los named directamente
const posts = await get("/posts");
```

## Re-exports (barriles)

Consolidan módulos en un solo punto de entrada:

```javascript
// 📁 utils/string.js
export function capitalize(str) {
  return str.charAt(0).toUpperCase() + str.slice(1);
}

// 📁 utils/number.js
export function clamp(num, min, max) {
  return Math.min(Math.max(num, min), max);
}

// 📁 utils/index.js — Barrel file
export { capitalize } from "./string.js";
export { clamp } from "./number.js";
export { default as Validador } from "./Validador.js";

// Ahora se importa desde un solo lugar
// 📁 app.js
import { capitalize, clamp, Validador } from "./utils/index.js";
```

## Dynamic imports (importaciones dinámicas)

Cargan módulos **bajo demanda** (lazy loading):

```javascript
// Se carga solo cuando el usuario hace click
boton.addEventListener("click", async () => {
  const { Chart } = await import("./chart.js");
  const chart = new Chart(datos);
  chart.render();
});

// Útil para code splitting y rendimiento
async function cargarPagina(nombre) {
  const modulo = await import(`./pages/${nombre}.js`);
  modulo.default.render();
}
```

## Módulos en HTML

```html
<!-- type="module" habilita import/export en el navegador -->
<script type="module" src="./app.js"></script>

<!-- Inline module -->
<script type="module">
  import { saludar } from "./utils.js";
  console.log(saludar("Samuel"));
</script>

<!-- nomodule — fallback para navegadores antiguos -->
<script nomodule src="./app-legacy.js"></script>
```

```
│   ├── api.js

├── 📁 utils/
│   ├── formatters.js
│   ├── validators.js
│   └── index.js
├── 📁 constants/
│   └── config.js

```
