# 📚 Arrays (Arreglos)

Un array es una **lista ordenada de elementos**. Es la estructura de datos más usada en JavaScript.

---

## Crear arrays

```javascript
// Literal (recomendado)
const frutas = ["🍎", "🍌", "🍊"];

// Constructor (menos común)
const numeros = new Array(1, 2, 3);

// Array vacío
const lista = [];

// Array con tipos mixtos
const mixto = ["texto", 42, true, null, { nombre: "JS" }, [1, 2]];
```

---

## Acceso y modificación

```javascript
const colores = ["rojo", "verde", "azul"];

// Acceso por índice (empieza en 0)
console.log(colores[0]); // "rojo"
console.log(colores[2]); // "azul"

// Último elemento
console.log(colores[colores.length - 1]); // "azul"
console.log(colores.at(-1));              // "azul" (ES2022) ✅

// Modificar
colores[1] = "amarillo";
console.log(colores); // ["rojo", "amarillo", "azul"]
```

## Métodos que MUTAN el array

```javascript
const arr = [1, 2, 3];

// push — Agrega al final
arr.push(4);         // [1, 2, 3, 4]

// pop — Elimina el último
arr.pop();           // [1, 2, 3]

// unshift — Agrega al inicio
arr.unshift(0);      // [0, 1, 2, 3]

// shift — Elimina el primero
arr.shift();         // [1, 2, 3]

// splice — Eliminar/insertar en posición específica
const letras = ["a", "b", "c", "d", "e"];
letras.splice(1, 2);           // Elimina 2 desde index 1 → ["a", "d", "e"]
letras.splice(1, 0, "x", "y"); // Inserta sin eliminar → ["a", "x", "y", "d", "e"]
letras.splice(1, 1, "z");      // Reemplaza 1 → ["a", "z", "y", "d", "e"]

// sort — Ordenar (muta!)
const nums = [3, 1, 4, 1, 5, 9];
nums.sort((a, b) => a - b); // Ascendente: [1, 1, 3, 4, 5, 9]
nums.sort((a, b) => b - a); // Descendente: [9, 5, 4, 3, 1, 1]

// reverse — Invertir
nums.reverse();
```

## Métodos que NO mutan (retornan nuevo array) ✅

### `map()` — Transformar cada elemento

```javascript
const numeros = [1, 2, 3, 4, 5];
const dobles = numeros.map(n => n * 2);
console.log(dobles); // [2, 4, 6, 8, 10]

// Ejemplo real
const usuarios = [
  { nombre: "Ana", edad: 25 },
  { nombre: "Luis", edad: 30 },
];
const nombres = usuarios.map(u => u.nombre);
console.log(nombres); // ["Ana", "Luis"]
```

### `filter()` — Filtrar elementos

```javascript
const edades = [12, 18, 22, 15, 30, 8];
const mayores = edades.filter(e => e >= 18);
console.log(mayores); // [18, 22, 30]

// Ejemplo: usuarios activos
const usuarios = [
  { nombre: "Ana", activo: true },
  { nombre: "Luis", activo: false },
  { nombre: "María", activo: true },
];
const activos = usuarios.filter(u => u.activo);
console.log(activos); // [{ nombre: "Ana"... }, { nombre: "María"... }]
```

### `reduce()` — Reducir a un solo valor

```javascript
const numeros = [1, 2, 3, 4, 5];

// Suma total
const suma = numeros.reduce((acumulador, actual) => acumulador + actual, 0);
console.log(suma); // 15

// Contar ocurrencias
const frutas = ["🍎", "🍌", "🍎", "🍊", "🍌", "🍎"];
const conteo = frutas.reduce((acc, fruta) => {
  acc[fruta] = (acc[fruta] || 0) + 1;
  return acc;
}, {});
console.log(conteo); // { "🍎": 3, "🍌": 2, "🍊": 1 }

// Agrupar por propiedad
const personas = [
  { nombre: "Ana", depto: "IT" },
  { nombre: "Luis", depto: "HR" },
  { nombre: "María", depto: "IT" },
];
const porDepto = personas.reduce((acc, p) => {
  (acc[p.depto] ??= []).push(p.nombre);
  return acc;
}, {});
console.log(porDepto); // { IT: ["Ana", "María"], HR: ["Luis"] }
```

### `find()` y `findIndex()`

```javascript
const usuarios = [
  { id: 1, nombre: "Ana" },
  { id: 2, nombre: "Luis" },
  { id: 3, nombre: "María" },
];

// find — Retorna el PRIMER elemento que cumple la condición
const usuario = usuarios.find(u => u.id === 2);
console.log(usuario); // { id: 2, nombre: "Luis" }

// findIndex — Retorna el ÍNDICE del primer elemento
const index = usuarios.findIndex(u => u.nombre === "María");
console.log(index); // 2
```

### `some()` y `every()`

```javascript
const notas = [85, 90, 70, 95, 60];

// some — ¿Alguno cumple?
const hayReprobado = notas.some(n => n < 70);
console.log(hayReprobado); // false

// every — ¿Todos cumplen?
const todosAprobados = notas.every(n => n >= 60);
console.log(todosAprobados); // true
```

### `flat()` y `flatMap()`

```javascript
// flat — Aplanar arrays anidados
const anidado = [1, [2, 3], [4, [5, 6]]];
console.log(anidado.flat());   // [1, 2, 3, 4, [5, 6]]
console.log(anidado.flat(2));  // [1, 2, 3, 4, 5, 6]
console.log(anidado.flat(Infinity)); // [1, 2, 3, 4, 5, 6]

// flatMap — map + flat(1)
const frases = ["hola mundo", "buenos días"];
const palabras = frases.flatMap(f => f.split(" "));
console.log(palabras); // ["hola", "mundo", "buenos", "días"]
```

## Spread operator con arrays

```javascript
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

// Combinar arrays
const combinado = [...arr1, ...arr2];
console.log(combinado); // [1, 2, 3, 4, 5, 6]

// Clonar un array (shallow copy)
const copia = [...arr1];
copia.push(99);
console.log(arr1);  // [1, 2, 3] — original intacto

// Insertar elementos
const conMedio = [...arr1.slice(0, 1), 1.5, ...arr1.slice(1)];
console.log(conMedio); // [1, 1.5, 2, 3]
```

## Destructuring de arrays

```javascript
const colores = ["rojo", "verde", "azul", "amarillo"];

const [primero, segundo] = colores;
console.log(primero, segundo); // "rojo" "verde"

// Saltar elementos
const [, , tercero] = colores;
console.log(tercero); // "azul"

// Rest
const [cabeza, ...resto] = colores;
console.log(cabeza); // "rojo"
console.log(resto);  // ["verde", "azul", "amarillo"]

// Swap (intercambiar) sin variable temporal
let a = 1, b = 2;
[a, b] = [b, a];
console.log(a, b); // 2, 1
```

## Encadenamiento de métodos (Method Chaining)

```javascript
const transacciones = [
  { concepto: "Venta", monto: 500 },
  { concepto: "Compra", monto: -200 },
  { concepto: "Venta", monto: 300 },
  { concepto: "Compra", monto: -150 },
  { concepto: "Venta", monto: 700 },
];

const totalVentas = transacciones
  .filter(t => t.monto > 0)             // Solo ventas
  .map(t => t.monto)                     // Extraer montos
  .reduce((sum, m) => sum + m, 0);       // Sumar

console.log(`Total ventas: $${totalVentas}`); // $1500
```

## 🏋️ Ejercicios

### Ejercicio 1 — Transformar datos
Dado un array de nombres, usa `map` para retornar un array donde cada nombre esté en mayúsculas.

### Ejercicio 2 — Filtrar y contar
Dado un array de edades `[12, 25, 8, 32, 17, 45, 16, 22]`, filtra los mayores de edad (≥18) y cuenta cuántos son.

### Ejercicio 3 — Reduce: carrito de compras
Dado un array de productos con `nombre` y `precio`, calcula el total usando `reduce`.

### Ejercicio 4 — Encadenar métodos
Dado un array de estudiantes con `nombre` y `nota`, filtra los aprobados (nota ≥ 60), extrae sus nombres con `map` y ordena alfabéticamente.

### Ejercicio 5 — Agrupar con reduce
Dado un array de palabras, agrúpalas por su primera letra usando `reduce`. El resultado debe ser un objeto: `{ a: ["ana", "antonio"], b: ["bruno"] }`.
